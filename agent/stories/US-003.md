# Historia de Usuario

## ID
HU-XXX

## Título
Consulta de hora local en país de destino

## Descripción
**Como** Usuario que envía remesas internacionales
**Quiero** Conocer la hora actual en el país de destino del envío
**Para** Conocer si es horario operativo en país destino.

## Criterios de Aceptación

### Escenario 1: Visualización de la hora al seleccionar país
```gherkin
Dado que el usuario se encuentra en la pàgina inicio 
Cuando selecciona un país de destino de la lista desplegable
Entonces el sistema debe mostrar la hora actual y la zona horaria de ese país en tiempo real
```

### Escenario 2: Cambio dinámico de país
```gherkin
Dado que ya hay un país seleccionado y se muestra una hora
Cuando el usuario cambia el país de destino a uno diferente
Entonces el sistema debe actualizar instantáneamente la hora para reflejar la del nuevo país
```

## Notas
 API: Se usara API externa de zonas horarias.

 UI/UX: El reloj estara cerca del campo País de destino.

 Dependencias: El listado de países debe tener mapeado su correspondiente código de zona horaria (ej: "America/Caracas").
